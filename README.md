# Recetas de cocina

## Propósito

Se trata de un repositorio demo en el que trabajaremos de manera colaborativa compartiendo nuestras recetas de cocina. Su propósito es demostrar el tema de las recetas de cocina, el fork y las pull requests.

## Cómo colaborar - reglas

Para colaborar lo que haremos será un fork del proyecto y pull request. Por favor, intentemos respestar los commits semánticos para que quede lo más homogéneo posible.

¡Gracias!
